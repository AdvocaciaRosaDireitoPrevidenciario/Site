# Rosa Advocacia e Consultoria - Site GitHub Pages

URL pública: https://advocaciarosadireitoprevidenciario.github.io/Site/

## Arquivos principais

- `index.html`: site completo em arquivo único, com CSS/JS embutidos e logo em base64.
- `google6ec71a1aa709502a.html`: arquivo de verificação do Google Search Console. Não apagar.
- `sitemap.xml`: sitemap para envio no Google Search Console.
- `robots.txt`: indicação do sitemap para rastreadores.
- `llms.txt`, `llm.txt`, `ai.txt`: resumo estruturado para leitura por ferramentas de IA.
- `sitemap.html`: mapa do site legível.
- `ror.xml`: mapa legado/compatibilidade.
- `404.html`: fallback para página não encontrada.

## Depois de publicar

No Google Search Console, enviar o sitemap:

```txt
https://advocaciarosadireitoprevidenciario.github.io/Site/sitemap.xml
```

Na inspeção de URL, solicitar indexação de:

```txt
https://advocaciarosadireitoprevidenciario.github.io/Site/
```
